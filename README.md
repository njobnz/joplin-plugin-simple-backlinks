# Simple Backlinks

Simple Backlinks is a plugin for Joplin that displays backlinks to the current note in the markdown viewer or separate panel. It is intended as drop-in replacement for the [Automatic Backlinks](https://discourse.joplinapp.org/t/automatic-backlinks-with-manual-insert-option/13632) plugin. For a more feature complete note referencing system consider using the [Joplin Note Link System](https://github.com/ylc395/joplin-plugin-note-link-system) plugin.
